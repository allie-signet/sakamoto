```
usage:
-> sakamoto! add_react [tag to trigger on]
(use :custom:[numerical emoji id] for server emoji)
-> sakamoto! add_embed [tag to trigger on]
(you can use https://glitchii.github.io/embedbuilder/ to get the embed json)
-> sakamoto! add_message [tag to trigger on]
-> sakamoto! add_template [tag to trigger on]
(templates use jinja2 syntax; as available through https://docs.rs/minijinja/latest/minijinja/index.html)
-> sakamoto! list_triggers
-> sakamoto! remove_trigger [tag]
```
